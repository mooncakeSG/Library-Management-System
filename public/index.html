Library Management System API

Available Endpoints

GET /api/members
Get all members
POST /api/members
Create a new member
GET /api/books
Get all books
POST /api/books
Create a new book
GET /api/borrowing
Get all borrowing records
POST /api/borrowing
Create a new borrowing record
GET /api/reservations
Get all reservations
POST /api/reservations
Create a new reservation
